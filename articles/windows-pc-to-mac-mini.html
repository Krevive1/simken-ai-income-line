AI副業を始めたとき、最初からMacを使っていたわけではありません。最初に使っていたのは、Windows 11を搭載した比較的低価格のノートパソコンでした。

このPCでChatGPTを使い、Webサイト制作や情報収集、GitHubの操作などを始めました。

その後、Flutterで制作したiPhoneアプリをApp Storeへ公開する段階で、私が選んだ開発方法ではmacOSとXcodeが必要になりました。そこでMac miniを追加購入しました。

WindowsノートPCが不要になったわけではなく、現在もブラウザ作業やサイト確認などを行うサブPCとして使っています。

この記事では、Windows PCでできたこと、Mac miniが必要になった理由、現在の使い分けを正直に整理します。

最初に結論

ChatGPTや一般的なWeb制作を始めるだけなら、Windows PCでも進められました。最初から高価なMacを買う必要はありませんでした。

一方で、iPhoneアプリをApp Storeへ公開する段階ではMac miniを追加しました。私が選んだFlutterとXcodeを使う開発方法では、最終的にMac環境が必要になったからです。

現在もWindows PCはサブ機として使用しています。ただし、これはすべての人に同じPCを推奨する記事ではありません。やりたい作業に応じて、必要な機材を選ぶことが重要です。

最初に使っていたWindowsノートPC

AI副業を始める時点で、最初から高価なMacを用意していたわけではありません。最初はWindows 11のノートパソコンを使い、ChatGPTでの調査や文章作成、Webサイトの確認、GitHubの操作などを進めていました。

このWindowsノートPCは、AI副業を始める初期作業では実際に使用できました。ブラウザで情報を調べ、ChatGPTに相談し、文章を書き、サイト表示を確認する作業には使えました。

ただし、高性能な動画編集や重い開発作業を目的に選んだ機種ではありません。できたことと、余裕が少なかったことを分けて考えています。

Windows PCでできたこと

Windows PCで進められた作業は、主にブラウザと軽い制作作業です。

  • ChatGPTの利用
  • 記事作成と情報収集
  • HTML・CSSなどのWeb制作
  • GitHubの確認や管理
  • 公開サイトの表示確認
  • 一般的な事務作業
  • ブラウザ上でのCloudflare管理
  • CursorやCodexを使った軽い制作・確認作業

これらは、最初の一歩としては十分に役立ちました。ただし、実際の性能以上に高性能なPCとして紹介するつもりはありません。重い処理を長時間行う用途では、余裕が少ない場面もあると考えています。

使って分かった良かった点

  • 最初から高額な機材を買わずに始められた
  • ChatGPTやブラウザ作業には使えた
  • 文章作成や軽いWeb作業には使えた
  • Mac mini購入後もサブPCとして利用できている
  • まず試してから、必要な機材を追加する判断ができた

最初からすべてをそろえようとすると、収益0円の段階で固定費や初期費用だけが増えます。自分の場合は、まずWindows PCで始めたことで、何に困るのか、どの段階で別の環境が必要になるのかを判断しやすくなりました。

使って分かった注意点

  • 高性能な動画編集や重い開発作業向けの機種ではない
  • 重いアプリを複数同時に使う用途では余裕が少ない可能性がある
  • CPU、メモリ、SSD容量は用途に合わせて確認する必要がある
  • iPhoneアプリをApp Storeへ公開する目的ではMac環境が別途必要になった
  • 商品ページの仕様、価格、在庫は変更される場合がある
  • すべての読者に同じ機種を推奨するものではない

PC選びでは、価格だけでなく用途を確認する必要があります。ブラウザ中心の作業なのか、動画編集をするのか、アプリ開発をするのかで、必要な性能は変わります。

Mac miniを追加購入した理由

その後、FlutterでiPhoneアプリを制作しました。最終的なiOSビルドではXcodeを使用し、iPhone実機で動作確認を行い、App Store Connectを使って申請・公開しました。

私が選んだFlutterとXcodeを使う開発方法では、最終的にMac環境が必要になりました。そのため、アプリ公開を目的としてMac miniを追加購入しました。

ここで大事なのは、「Windowsでは絶対にiPhoneアプリを作れない」と単純に断定することではありません。自分が選んだ開発方法と公開工程では、macOSとXcodeが必要になったという整理です。

現在のWindows PCとMac miniの使い分け

現在は、WindowsノートPCとMac miniを用途で分けています。

WindowsノートPC

  • ブラウザでの調査
  • サイトの表示確認
  • 文章作成
  • 管理画面の確認
  • サブPCとして使用

WindowsノートPCは、現在もブラウザ作業や確認作業に使っています。Mac miniを買ったからといって、Windows PCが無駄になったわけではありません。

Mac mini

  • Flutter開発
  • Xcodeでのビルド
  • iPhone実機確認
  • App Storeへの申請
  • Codex、Cursor、Web制作の主な作業

Mac miniは、iPhoneアプリ公開に必要な作業と、現在の主な制作作業で使っています。用途を誇張せず、必要になった段階で追加した機材として整理しています。

最初からMacを買うべき人

次のような人は、最初からMacを検討してもよいと思います。

  • iPhoneアプリを開発し、App Storeへ公開する予定がある
  • Xcodeを使う必要がある
  • macOS専用ソフトを使う
  • Apple製品との連携を重視する

特にApp Store公開まで見据えている場合は、開発方法と必要な環境を先に確認した方が遠回りを減らせます。

Windows PCから始めてもよい人

一方で、次のような段階ならWindows PCから始めてもよいと感じています。

  • ChatGPTを使って記事や情報を作りたい
  • HTML・CSSなどのWeb制作を試したい
  • GitHubやCloudflareを使ってサイトを公開したい
  • まず低予算で副業を試したい
  • iPhoneアプリ開発を現時点で予定していない

最初の目的が「AIで何か作ってみる」「サイトを公開してみる」なら、必ずしも最初から高価な環境をそろえる必要はありません。

重要だったのは最初から最高のPCを買うことではなかった

重要だったのは、最初から一番高い機材を買うことではなく、実際に作業を始め、やりたいことが変わった段階で必要な機材を追加したことです。

Windows PCで始めた経験が無駄になったわけではなく、現在もサブPCとして活用しています。

AI副業の機材選びは、他人のおすすめをそのまま買うより、自分が何を作りたいのか、どこまで公開したいのかを基準に考える方が現実的です。
