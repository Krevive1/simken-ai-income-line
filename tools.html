使用ツール

使っていないサービスのランキングや根拠のないおすすめは作りません。実際の用途、良かった点、注意点を整理します。

AI・文章作成

ChatGPT

用途: 構成整理、文章化、アイデア出し。

良かった点: 初心者でも考えを言語化しやすい。

注意点: 仕様や料金は変わるため公式情報の確認が必要。

Cursor

GitHub / Cloudflare Pages

用途: ファイル管理と静的サイト公開。

良かった点: HTMLサイトをそのまま公開しやすい。

注意点: 接続設定やデプロイエラーの読み方に慣れが必要。

自動化

n8n

用途: SNS投稿や定型作業の自動化検証。

良かった点: 処理の流れを目で確認しやすい。

注意点: 認証、API、投稿品質は完全放置にしにくい。

スマートフォン

iPhone

用途: アプリ確認、スマホ表示確認。

良かった点: 実際の読者に近い環境で確認できる。

注意点: PCだけで見ていると読みにくさを見落とす。

パソコン

WindowsノートPC

用途: AI副業を始めた当初に使用。現在もブラウザ作業やサイト確認などを行うサブPCとして使用しています。

良かった点: ChatGPT、文章作成、軽いWeb作業、GitHub確認などの初期作業を進められた。

注意点: 重い開発作業やiPhoneアプリのApp Store公開では、用途に応じて別の環境が必要になる場合があります。
